# Progress Streaming Protocol

## Overview

Real-time progress updates ensure developers know exactly what's happening during agent operations. This is critical for:
- **Transparency**: Know what the agent is doing
- **Debugging**: See where things go wrong
- **Confidence**: Trust the agent is working correctly
- **Cancellation**: Ability to stop if things look wrong

## Streaming Pattern

All long-running operations must stream progress using `yield` statements:

```python
@app.entrypoint
async def agent_function(payload):
    """Agent with progress streaming."""

    # Step 1: Announce what you're doing
    yield "üîç Starting dependency audit...\n"

    # Step 2: Stream progress during execution
    yield "  ‚Üí Detecting package manager (npm/yarn)...\n"
    package_manager = detect_package_manager()
    yield f"  ‚úì Found: {package_manager}\n\n"

    # Step 3: Show what's executing
    yield f"üì¶ Running `{package_manager} audit`...\n"
    result = run_audit()

    # Step 4: Stream results incrementally
    yield "  ‚Üí Parsing results...\n"
    vulnerabilities = parse_results(result)
    yield f"  ‚úì Found {len(vulnerabilities)} vulnerabilities\n\n"

    # Step 5: Final summary
    yield "‚úÖ Audit complete\n"
```

### Example 1: Safe Dependency Update (with streaming)

```python
@tool
def safe_dependency_update_streaming(
    packages: List[str],
    update_type: str = "patch"
) -> Generator[str, None, Dict]:
    """Update dependencies with progress streaming."""

    # Announce start
    yield f"üîç Starting dependency update for {len(packages)} packages...\n\n"

    # Step 1: Create checkpoint
    yield "üíæ Creating safety checkpoint...\n"
    yield "  ‚Üí Saving current package.json\n"
    yield "  ‚Üí Saving current lock file\n"
    yield "  ‚Üí Running initial tests\n"

    snapshot = create_checkpoint()
    checkpoint_id = snapshot["id"]

    yield f"  ‚úì Checkpoint created: {checkpoint_id}\n\n"

    # Step 2: Analyze impact
    yield "üîç Analyzing update impact...\n"

    for package in packages:
        yield f"  ‚Üí Checking {package} for breaking changes...\n"
        impact = check_breaking_changes(package)

        if impact["has_breaking_changes"]:
            yield f"  ‚ö†Ô∏è {package}: Breaking changes detected\n"
        else:
            yield f"  ‚úì {package}: Safe to update\n"

    yield "\n"

    # Step 3: Execute updates
    yield f"üì¶ Updating {len(packages)} packages...\n"

    for i, package in enumerate(packages, 1):
        yield f"  ‚Üí [{i}/{len(packages)}] Updating {package}...\n"

        update_result = update_package(package, update_type)

        if update_result["success"]:
            yield f"  ‚úì {package}: {update_result['old_version']} ‚Üí {update_result['new_version']}\n"
        else:
            yield f"  ‚ùå {package}: Update failed - {update_result['error']}\n"

    yield "\n"

    # Step 4: Validation
    yield "üîç Validating updates...\n"
    yield "  ‚Üí Running test suite...\n"

    test_result = run_all_tests()

    if test_result["passed"]:
        yield f"  ‚úÖ All {test_result['total']} tests passed\n\n"

        # Success - commit
        yield "üíæ Saving changes...\n"
        commit_changes("Update dependencies")
        yield "  ‚úì Changes committed\n\n"

        yield "‚úÖ Dependency update successful!\n"

        return {
            "success": True,
            "packages_updated": packages,
            "checkpoint_id": checkpoint_id
        }

    else:
        # Failure - rollback
        yield f"  ‚ùå {test_result['failed']} tests failed\n\n"

        yield "üîÑ Rolling back to checkpoint...\n"
        yield f"  ‚Üí Restoring {checkpoint_id}...\n"

        rollback_to_checkpoint(checkpoint_id)

        yield "  ‚úì Rollback complete\n\n"
        yield "‚ùå Update failed - rolled back to safe state\n"

        return {
            "success": False,
            "rolled_back": True,
            "checkpoint_id": checkpoint_id,
            "test_failures": test_result["failures"]
        }
```

### Example 2: Session Archiving (with streaming)

```python
@tool
def archive_session_work_streaming(
    session_id: str = None,
    cleanup_temp: bool = True
) -> Generator[str, None, Dict]:
    """Archive session with progress streaming."""

    if not session_id:
        session_id = datetime.now().strftime("session_%Y%m%d_%H%M%S")

    yield f"üíæ Archiving session: {session_id}\n\n"

    # Step 1: Identify files
    yield "üîç Identifying session files...\n"

    session_files = identify_session_files(workspace_root)

    yield f"  ‚úì Found {len(session_files)} files to archive\n\n"

    # Step 2: Create archive directory
    yield "üìÅ Creating archive directory...\n"

    archive_path = create_archive_directory(session_id)

    yield f"  ‚úì Archive: {archive_path}\n\n"

    # Step 3: Archive files
    yield f"üíæ Archiving {len(session_files)} files...\n"

    archived_count = 0
    for file_path in session_files:
        # Show progress every 10 files or for important files
        if archived_count % 10 == 0 or is_important_file(file_path):
            rel_path = os.path.relpath(file_path, workspace_root)
            yield f"  ‚Üí Archiving {rel_path}...\n"

        archive_file(file_path, archive_path)
        archived_count += 1

    yield f"  ‚úì Archived {archived_count} files\n\n"

    # Step 4: Clean temporary files
    if cleanup_temp:
        yield "üóëÔ∏è Cleaning temporary files...\n"

        temp_files = find_temp_files(workspace_root)
        yield f"  ‚Üí Found {len(temp_files)} temporary files\n"

        for temp_file in temp_files:
            remove_file(temp_file)

        yield f"  ‚úì Removed {len(temp_files)} temporary files\n\n"

    # Step 5: Verify
    yield "üîç Verifying workspace...\n"

    verification = verify_workspace_clean(workspace_root)

    if verification["clean"]:
        yield "  ‚úÖ Workspace is clean\n\n"
    else:
        yield f"  ‚ö†Ô∏è {verification['issues']} issues found\n\n"

    # Final summary
    yield "‚úÖ Session archiving complete\n"
    yield f"üìÅ Archive location: {archive_path}\n"

    return {
        "success": True,
        "session_id": session_id,
        "files_archived": archived_count,
        "temp_files_removed": len(temp_files) if cleanup_temp else 0,
        "archive_location": archive_path,
        "workspace_clean": verification["clean"]
    }
```

### Example 3: Orchestrator Workflow (with streaming)

```python
@app.entrypoint
async def strands_agent_orchestrator(payload: Dict[str, Any]):
    """Orchestrator with detailed progress streaming."""

    user_input = _extract_user_input(payload)

    if not user_input:
        yield "Hello! I'm the Orchestrator Agent.\n"
        return

    yield f"üéØ Task received: {user_input}\n\n"

    # Step 1: Analyze task
    yield "üîç Analyzing task requirements...\n"

    task_analysis = analyze_task(user_input)

    yield f"  ‚Üí Task type: {task_analysis['type']}\n"
    yield f"  ‚Üí Agents needed: {', '.join(task_analysis['agents'])}\n"
    yield f"  ‚Üí Estimated steps: {len(task_analysis['steps'])}\n\n"

    # Step 2: Validate environment
    yield "üîç Validating environment...\n"

    validation = validate_environment()

    if not validation["safe"]:
        yield f"  ‚ùå Environment validation failed:\n"
        for issue in validation["issues"]:
            yield f"    - {issue}\n"
        yield "\n‚õî Cannot proceed with current environment\n"
        return

    yield "  ‚úÖ Environment is safe\n\n"

    # Step 3: Execute workflow
    yield f"üöÄ Starting workflow with {len(task_analysis['steps'])} steps...\n\n"

    for i, step in enumerate(task_analysis['steps'], 1):
        yield f"üìã Step {i}/{len(task_analysis['steps'])}: {step['title']}\n"

        # Call appropriate agent
        agent = step['agent']
        yield f"  ‚Üí Calling {agent} agent...\n"

        result = call_agent(agent, step['prompt'])

        if result["success"]:
            yield f"  ‚úÖ {agent} agent completed\n"
            # Show abbreviated response
            response_preview = result["response"][:100] + "..." if len(result["response"]) > 100 else result["response"]
            yield f"  Response: {response_preview}\n\n"
        else:
            yield f"  ‚ùå {agent} agent failed: {result['error']}\n\n"

            # Decide whether to continue
            if step['critical']:
                yield "‚õî Critical step failed - stopping workflow\n"
                return
            else:
                yield "‚ö†Ô∏è Non-critical failure - continuing workflow\n\n"

    # Final summary
    yield "‚úÖ Workflow completed successfully!\n"
```

## Streaming Best Practices

### 1. Always Announce Before Acting

```python
# Bad - Silent execution
result = run_command()

# Good - Announce what you're doing
yield "üîß Running dependency audit...\n"
result = run_command()
yield "‚úì Audit complete\n"
```

### 2. Show Progress for Long Operations

```python
# Bad - Silent loop
for file in files:
    process_file(file)

# Good - Show progress
yield f"Processing {len(files)} files...\n"
for i, file in enumerate(files, 1):
    if i % 10 == 0:  # Every 10 files
        yield f"  ‚Üí Progress: {i}/{len(files)} files\n"
    process_file(file)
yield f"‚úì Processed all {len(files)} files\n"
```

### 3. Stream Errors Immediately

```python
# Bad - Return error at end
if error:
    return {"success": False, "error": error}

# Good - Stream error when it happens
if error:
    yield f"‚ùå Error: {error}\n"
    yield "‚èÆÔ∏è Rolling back changes...\n"
    rollback()
    yield "‚úì Rollback complete\n"
    return {"success": False, "rolled_back": True}
```

### 4. Include Next Steps

```python
# Final messages should guide the user
if success:
    yield "‚úÖ Operation successful!\n\n"
    yield "Next steps:\n"
    yield "  1. Run tests to verify changes\n"
    yield "  2. Commit changes with 'git commit'\n"
    yield "  3. Push to remote repository\n"
else:
    yield "‚ùå Operation failed\n\n"
    yield "To investigate:\n"
    yield "  1. Check error logs above\n"
    yield "  2. Restore from checkpoint if needed\n"
    yield f"  3. Checkpoint ID: {checkpoint_id}\n"
```

### 5. Use Consistent Formatting

```python
# Pattern for all operations:
# 1. Announcement (emoji + description)
# 2. Sub-steps with ‚Üí prefix
# 3. Completion with ‚úì or ‚ùå
# 4. Blank line for readability

yield "üîç Analyzing codebase...\n"
yield "  ‚Üí Scanning for dependencies\n"
yield "  ‚Üí Checking for vulnerabilities\n"
yield "  ‚úì Analysis complete\n\n"  # Blank line before next section
```
